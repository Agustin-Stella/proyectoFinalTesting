# Proyecto Final - Testing Automatizado con Java, Selenium y Cucumber

Este proyecto es una automatización de pruebas realizada como práctica final de mi formación en Testing. Utiliza Java, Selenium WebDriver y Cucumber.  El objetivo es validar funcionalidades clave de una página web.

## Sitio web bajo prueba

https://automationexercise.com/

## ✅ Funcionalidades cubiertas

-Crear cuenta
-Login
-Verificar el Home
-Añadir produtos al carrito
-Buscar un producto por su nombre
